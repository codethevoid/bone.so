## Examples

The following variants are available for the Button component:

### Primary

Primary buttons are the default style and are used for main actions.

<Showcase>
  <Button>Primary</Button>
</Showcase>

<CodeBlock filename={'primary-btn.tsx'} code={`import { Button } from "@bonekit/ui/button";

  export const PrimaryBtn = () => {
    return <Button>Primary</Button>;
  };`} />

### Soft
Soft buttons have a subtle background and are used for less prominent actions.

<Showcase>
  <Button variant="soft">Soft</Button>
</Showcase>

<CodeBlock filename={'soft-btn.tsx'} code={`import { Button } from "@bonekit/ui/button";

  export const SoftBtn = () => {
    return <Button variant="soft">Primary</Button>;
  };`} />

### Outline
Outline buttons have a border and are used for secondary actions.

<Showcase>
  <Button variant="outline">Outline</Button>
</Showcase>

<CodeBlock filename={'outline-btn.tsx'} code={`import { Button } from "@bonekit/ui/button";

  export const OutlineBtn = () => {
    return <Button variant="outline">Outline</Button>;
  };`} />

### Ghost
Ghost buttons have no background or border and are used for minimalistic designs.

<Showcase>
  <Button variant="ghost">Ghost</Button>
</Showcase>

<CodeBlock filename={'ghost-btn.tsx'} code={`import { Button } from "@bonekit/ui/button";

  export const GhostBtn = () => {
    return <Button variant="ghost">Ghost</Button>;
  };`} />

### Danger
Danger buttons are used for destructive actions and have a red color scheme.

<Showcase>
  <Button variant="danger">Danger</Button>
</Showcase>

<CodeBlock filename={'danger-btn.tsx'} code={`import { Button } from "@bonekit/ui/button";

  export const DangerBtn = () => {
    return <Button variant="danger">Danger</Button>;
  };`} />

### Custom
You can apply custom styles using Tailwind CSS classes.

<Showcase>
  <Button
    className="bg-blue-600 text-white hover:bg-blue-700 shadow-blue-600/20 hover:scale-[103%] ease focus:ring-0 active:scale-[98%]">Custom</Button>
</Showcase>

<CodeBlock filename="custom-btn.tsx" code={`import { Button } from "@bonekit/ui/button";

  export const CustomBtn = () => {
    return (
      <Button className="bg-blue-600 text-white hover:bg-blue-700 shadow-blue-600/20 hover:scale-[103%] ease focus:ring-0 active:scale-[98%]">
        Custom
      </Button>
    );
  };`} />

### Link

You can make any element look like the button component by using the `passThrough` prop. This is useful for creating links that look like buttons.

<Showcase>
  <Button passThrough>
    <NextLink href="https://github.com/codethevoid/bone.so" target="_blank" rel="noopener noreferrer">
      <span>View Github</span>
    </NextLink>
  </Button>
</Showcase>

<CodeBlock filename="link-btn.tsx" code={`import { Button } from "@bonekit/ui/button";
import NextLink from "next/link";

  export const LinkBtn = () => {
    return (
      <Button passThrough>
        <NextLink href="https://github.com/codethevoid/bone.so" target="_blank" rel="noopener noreferrer">
          View Github
        </NextLink>
      </Button>
    );
  };`} />
